## 📱 Utilisation sur Mobile

### Layout Responsive

Le portfolio s'adapte automatiquement à tous les écrans :

```
Desktop (1200px+)
└─ Layout large
   ├─ 2 colonnes (profil + info)
   ├─ Dashboards côte à côte
   └─ Textes larges

Tablet (768-1024px)
└─ Layout moyen
   ├─ 1 colonne (empilé)
   ├─ Dashboards séquentiels
   └─ Menus adaptés

Mobile (< 768px)
└─ Layout compact
   ├─ Menu hamburger
   ├─ Stack vertical
   ├─ Boutons tactiles
   └─ Textes optimisés
```

### Menu Mobile

**Ouverture :**
1. Cliquer icône hamburger (3 lignes en haut droit)
2. Menu glisse depuis le côté
3. Cliquer un lien = ferme menu automatiquement

**Navigation :**
- Tous les liens disponibles
- Même contenu qu'ordinateur
- Optimisé pour doigts (hitbox +)

### Tableaux de Bord sur Mobile

**Power BI Mobile :**
- Installer app "Power BI" (App Store / Google Play)
- Se connecter avec compte Microsoft
- Accéder aux rapports depuis le cloud

**Limitation :** Fichiers `.pbix` locaux ne s'ouvrent pas sur mobile. Solutions :

1. **Publier sur Power BI Online :**
   - Ouvrir fichier dans Power BI Desktop
   - Publish → Cloud
   - Accéder via app mobile

2. **Voir PDF statique :**
   - Télécharger TB_Profil.pdf / TB_Veille.pdf
   - Ouvrir dans lecteur PDF
   - Vue statique (pas d'interactivité)

### Conseils Mobile

✅ **À faire :**
- Utiliser Wi-Fi (fichiers peuvent être volumineux)
- Portrait ou paysage selon contenu
- Zoomer si texte trop petit (pinch)
- Utiliser menu hamburger

❌ **À éviter :**
- Données mobiles (peut être lent)
- Écran faible luminosité (textes fins)
- Faire défiler trop vite (animations à voir)

## 🔧 Troubleshooting

### Problème : Portfolio web ne charge pas

**Causes possibles :**
- Pas de connexion internet
- Navigateur obsolète
- Cache navigateur corrompu

**Solutions :**
1. Vérifier connexion internet
2. Rafraîchir page : Ctrl+F5 (Windows) / Cmd+Shift+R (Mac)
3. Vider cache :
   - Ctrl+Shift+Delete (Windows)
   - Cmd+Shift+Delete (Mac)
4. Essayer navigateur différent
5. Vérifier URL correcte : https://assiakamoune-coder.github.io

---

### Problème : Fichier `.pbix` ne s'ouvre pas

**Causes :**
- Power BI Desktop pas installé
- Version Power BI obsolète
- Fichier corrompu

**Solutions :**
```bash
1. Installer Power BI Desktop :
   https://powerbi.microsoft.com/fr-fr/downloads/

2. Vérifier version à jour :
   Aide → À propos → Version

3. Si corrompu, re-télécharger fichier

4. Si erreur persiste :
   → Contacter assia.kamoune@gmail.com
```

---

### Problème : Menu hamburger ne fonctionne pas

**Cause :** JavaScript désactivé ou non chargé

**Solutions :**
1. Vérifier JavaScript activé :
   - Paramètres navigateur
   - Chercher "JavaScript"
   - Activer si désactivé

2. Rafraîchir page

3. Essayer navigateur différent

---

### Problème : Formulaire contact ne fonctionne pas

**Cause :** JavaScript ou email client non configuré

**Solutions :**
1. Utiliser contact direct :
   - Email : assia.kamoune@gmail.com
   - Tél : +33 6 59 04 49 22
   - LinkedIn : messagerie directe

2. Si formulaire nécessaire :
   - Vérifier JavaScript activé
   - Vérifier client email configuré (Outlook, Gmail, etc.)

---

### Problème : Les animations sautent / scintillent

**Cause :** Préférence réduction mouvement activée

**Solutions :**
1. Windows : Paramètres > Accessibilité > Affichage > Réduire mouvement (OFF)
2. Mac : Préférences Système > Accessibilité > Affichage > Réduire mouvement (OFF)

---

### Problème : Textes illisibles sur mobile

**Causes :**
- Luminosité écran trop faible
- Zoom trop grand/petit
- Résolution incompatible

**Solutions :**
1. Augmenter luminosité écran
2. Pinch/zoom : deux doigts écartés
3. Orienter portrait/paysage
4. Agrandir texte système

---

### Problème : Liens externes ne fonctionnent pas

**Causes :**
- Pas de connexion internet
- URL morte (lien obsolète)

**Solutions :**
1. Vérifier connexion internet
2. Copier URL manuellement dans navigateur
3. Si lien toujours mort :
   - Contacter par autre moyen
   - Email : assia.kamoune@gmail.com
